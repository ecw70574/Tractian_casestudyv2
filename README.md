# Tractian_casestudyv2

## Setup

1. Clone the repo

git clone https://github.com/ecw70574/Tractian_casestudyv2.git

2. Create a `.env` file in the root directory

3. Add your API keys
LINKDAPI_KEY = [your key here]
GEOAPIFY_KEY = [your key here]

Make LinkdAPI account (free) here: https://linkdapi.com
Make GeoApify account (free) here: https://www.geoapify.com

4. Install dependencies

pip install -r requirements.txt
